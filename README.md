# 异步加载百度地图或高德地图

### 运行项目
```shell
git clone https://github.com/aries-dp/vue-maps.git

cd vue-maps
yarn 
yarn serve

```
### 注意
在utils文件夹中keys.js需要申请高德地图或百度地图对应的公钥
